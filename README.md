# Pharmacy Marketplace Onboarding Project (Chowdeck)

## Project Overview

This project documents the onboarding of a community pharmacy onto the Chowdeck delivery marketplace. The goal was to create a structured digital storefront that allows customers browse and order pharmacy products online.


**Objective:** Successfully configured and deployed a community pharmacy on the Chowdeck marketplace.

**Live Storefront:**  
https://store.chowdeck.com/alausa/supermarkets/denicare-pharmacyalausa8czdkq

## Responsibilities

- Registered pharmacy store on Chowdeck platform
- Created structured product categories
- Uploaded product listings
- Added pricing information
- Uploaded product images
- Configured store opening and closing schedule
- Prepared store for marketplace launch readiness
